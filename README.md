# CulinaryHub — Modern Website Cafe & Restaurant

## Overview

**CulinaryHub** adalah project website modern untuk cafe, restaurant, coffee shop, dan bisnis F&B yang dibuat menggunakan [React](https://react.dev?utm_source=chatgpt.com) dengan fokus pada:

* visual experience,
* branding presentation,
* digital menu interaction,
* online reservation,
* dan user experience yang immersive.

Project ini dirancang sebagai dummy project premium untuk portfolio agency agar terlihat seperti website restoran modern kelas atas yang benar-benar siap dipakai client.

---

Project ini dibuat untuk menunjukkan kemampuan:
